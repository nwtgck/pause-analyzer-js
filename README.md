# Pause Analyzer

## What is this?

This analyzer adds slashs in pauses.


### Example 1

```
I have a pen.
```
=>

```
I have a pen.///
```


### Example 2

```
When I had a pen, I had also an apple.
```
=>

```
When I had a pen,// I had also an apple./// 
```
